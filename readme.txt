

# Cleaning Agency Website  

## Overview  
This is a responsive, client-focused website for a cleaning agency. Built with **HTML**, **CSS**, and **JavaScript**, it showcases services, includes a booking form, and provides an engaging user experience.  

---

## Features  
- **Homepage**:  
  - Highlights the agency’s key services and benefits.  
  - Call-to-action (CTA) buttons for bookings and inquiries.  

- **Services Page**:  
  - Detailed descriptions of cleaning services with clear, user-friendly layouts.  

- **Booking Form**:  
  - Interactive form for scheduling cleaning services.  
  - Validations implemented using JavaScript.  

- **Contact Page**:  
  - Easy-to-use contact form.  
  - Includes phone number, email, and social media links.  

- **Responsive Design**:  
  - Fully optimized for desktop, tablet, and mobile devices using CSS media queries.  

---

## Technologies Used  
- **HTML5**: Structuring content.  
- **CSS3**: Styling and layout, including Flexbox and Grid for responsiveness.  
- **JavaScript**: Interactivity and form validation.  

---

## Installation  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/your-username/cleaning-agency-website.git  
   ```  
2. Navigate to the project folder:  
   ```bash  
   cd cleaning-agency-website  
   ```  
3. Open `index.html` in your browser to view the website.  

---

## Usage  
1. Explore the homepage for an overview of services.  
2. Navigate to the **Services Page** to learn about available cleaning options.  
3. Use the **Booking Form** to schedule a cleaning session.  
4. Contact the agency via the **Contact Page** for additional inquiries.
